HTML

HyperText Markup Language, is the standard markup language used to create web pages. It uses a system of tags and elements to define the layout and components of a webpage, such as headings, paragraphs, images, and links.

CSS

which stands for Cascading Style Sheets, is a style sheet language used to describe the presentation of a document written in HTML. It controls the visual aspects of a webpage, such as the colors, fonts, and layout, making the design and aesthetics separate from the content.

JavaScript

JavaScript is a versatile programming language that adds interactivity and dynamic behavior to websites. It allows developers to create features like animations, interactive forms, and real-time updates directly within a user's web browser.

WordPress

WordPress is a free, open-source content management system (CMS) that allows users to create and manage websites easily without needing to code from scratch. It is widely used for building everything from simple blogs to complex business websites, thanks to its extensive library of themes and plugins.

Canva

Canva is an easy-to-use online graphic design platform that helps people create professional-looking visual content like social media posts, presentations, and posters. Its drag-and-drop interface and huge library of templates and design elements make it accessible for anyone, regardless of their design skills.

SEO

Search Engine Optimization, is the process of improving a website's visibility and ranking on search engines like Google and Bing. By optimizing content and technical elements, SEO aims to drive more organic, or unpaid, traffic to a website.
